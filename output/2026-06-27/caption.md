# 6/27 小红书文案（内容＝6/26 · 头条 Claude Tag）

## 标题（推荐 · 12字）
Claude现在能当你同事了

## 正文（不带 emoji）
今天 AI 圈 5 件大事，1 分钟看完。

一、Anthropic 推出 Claude Tag，让 Claude 像同事一样住进公司的 Slack 群，有自己的名字和记忆。群里 @它一下把活交过去，它就分几步自己做、边做边汇报进度。最猛的是，Anthropic 自家团队靠它写出了 65% 的代码，连 Claude Tag 自己也大半是它写的。
二、OpenAI 下一代旗舰 GPT-5.6 要发布，可这回得先过白宫。特朗普政府以安全为由，要它先只开放给一小撮政府批准的合作方、逐个放行。这是美国政府头一回在模型发布前就主动出手限制本国 AI 公司。
三、Notion 把自家邮箱应用关掉了，原因有点反常：一多半用户压根不开收件箱，全靠 agent 替自己收发邮件，公司干脆说要全押在 agent 上。
四、一篇论文发现，大模型最后几层为了把话说通顺，会把本来算对的答案悄悄改错；提前在它已经有把握的那层取答案反而更准，还不用重训。
五、另一篇论文发现，训会用命令行的 AI，真正卡人的是没有好题库和数据，算法反倒不难；把训练环境造好，一个九十亿参数的小模型就反超了更大的模型。
