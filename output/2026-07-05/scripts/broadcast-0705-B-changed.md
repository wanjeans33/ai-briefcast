一分钟带你了解AI圈的新鲜事。

先说阿里把 Claude 请出了门。据多家媒体报道，阿里巴巴七月三日发内部通知，全面禁用 Anthropic 旗下全部产品，包括编程工具 Claude Code，七月十日生效，全员卸载。这事有前情。Anthropic 去年九月起就不让中资控股公司用 Claude。上个月底它又写信给美国参议院和白宫，指控阿里千问用两万五千个假账号蒸馏 Claude 的数据，但至今没拿出公开证据。之后中国用户遭遇大面积封号，Claude Code 里还被发现藏着检测中国用户的机制。先封禁，再指控，又暗中设防，阿里干脆不用了。

再看一个反差。美国一家卖三明治的连锁店要上市，招股书里都郑重写上了 AI。可据报道，扎克伯格在 Meta 内部会议上向员工承认，AI agent 的进展没有他预期的那么快。外面吹得越响，里面越清楚活还没干完。

OpenAI 这边在跟政府套近乎。据金融时报报道，OpenAI 提出探讨把公司百分之五的股权交给美国政府，换取跟特朗普政府缓和关系。Anthropic 也刚靠让步换来 Fable 5 重新上线。不过股权这事目前只是设想，连时间表都没有。AI 竞争早就不只是技术竞争了。

论文这边，耶鲁大学想让 AI 学会真心说我不知道。对付 AI 一本正经地胡说，常规做法是在外面加一层核查。这篇论文换了个思路，把知道自己不知道这件事直接当成训练目标，用强化学习去训，让模型嘴上说的我不确定，真的对应它心里的没把握。论文报告这个能力最多提升六成多，还不牺牲准确率。敢说不知道的 AI，反而更让人放心。

另一篇论文泼了盆冷水。训练能干长活的 AI，很多人给它的每个中间步骤打分，方法一个比一个花哨。作者做了一千二百多次实验，把二十一种打分方法拉到一起体检，结果最简单的办法，直接拿提示词去问模型，稳定打赢了多数复杂方法。花哨不等于有效，动手之前先量一量，比跟风上新招更省钱。
