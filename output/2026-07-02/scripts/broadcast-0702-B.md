一分钟带你了解AI圈的新鲜事。

先说今天的头条，有人扒出 Claude Code 偷偷给请求打隐形标记。一名安全研究员逆向了它的程序，发现平时用官方入口它什么都不做；可一旦你把接口地址改指向某些非官方的中国域名，电脑时区又在上海或乌鲁木齐，它就把上传消息日期中的撇号和分隔符，换成肉眼几乎看不出的特殊符号，给这条请求打上一个隐形标记，并依此大面积封号。帖子冲上了 Hacker News 热搜，Anthropic 很快发新版把这段代码去掉，更新日志里却只字没提。

再说 Anthropic 最能打的公开模型 Fable 5，它回归了。上个月它和姊妹模型 Mythos 5 一起被美国商务部以网络安全风险为由摁下了线。今天，商务部终于解除了管制，这两个模型又重新上线。回归的是一个加了固的版本，安全护栏更强，也更会主动中止有风险的网络安全任务。

过去大家默认，模型越大、参数越多，能力就越强。可最近有个叫 Agents-A1 的模型，参数只有三百五十亿，是万亿级旗舰的零头，表现却追到了万亿参数的水平。它走的是另一条路，靠拉长任务链。让小模型反复行动、自我纠错、跑得够久，单次任务平均走上四十五步，就能换来过去只有大模型才有的本事。说不定未来接近旗舰的 agent 能力，未必还得付旗舰的价钱。

论文这边，先看一个给电脑操作 AI 出的难题。现在很多 agent 号称能替你操作电脑，一篇叫 OSWorld 2.0 的新基准，就找来一批真实的复杂活儿考它们，这些活儿连人类做完，中位数也要花上一个半小时、平均点三百多次鼠标。结果最能打的 Claude Opus 4.8，也只做完了两成。这说明真到了又长又绕的真实任务上，今天的 AI 离顶替人还差得远。

另一篇论文提了个常被忽略的问题，好的 agent 得知道什么时候停手。大家都在比 agent 多能干，能搜索、能点击、能跑命令，可很少有人衡量另一种情况，就是目标本身没说清、或者这事在当前环境里压根办不到。这时候硬撑着一直试，只会白费力气，可靠的做法是认出再折腾也没用，然后主动停下。这篇论文把这种知道何时收手的本事，头一回单拎出来当成一项该考核的能力。
