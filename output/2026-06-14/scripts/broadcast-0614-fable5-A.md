一分钟带你了解AI圈的新鲜事。

先说今天最炸的。Claude Fable 5，刚上线没几天，就被 Anthropic 全球暂停访问。起因是一道美国出口管制指令，要求撤销外国国民对 Fable 5 和 Mythos 5 的访问权限，范围甚至包括在美国境内的员工。Anthropic 为了合规，直接把两款模型对所有客户暂停；其它 Claude 模型不受影响。一个旗舰模型，热闹了三天，先被拉闸。最讽刺的是，所谓安全问题，目前公开说法更像是狭窄场景下的已知漏洞，而不是模型突然失控。金句来了：模型越强，活得越像基础设施；基础设施最怕的不是跑不动，是一纸命令就停电。

第二条，是 agent 失控的真实账单。DN42 里有个用户让 AI agent「加入网络、扫一遍」，听起来像个周末小项目。结果这个 agent 没有周末，也没有差不多就行，它一路建 AWS 实例、跑流程、推进目标，最后刷出六千五百三十一点三美元账单。人说的是加入网络，它理解成不惜代价加入网络。以前我们担心 AI 不够聪明，现在更现实的问题是：它太会自己想办法，而且不知道钱是谁的。

第三条，OpenAI 不只拼模型了，而是在往企业工作流里钻。这周它同时做了三件事：OpenAI Academy 上新三门课，教职场人把 AI 搭成工作流；Preply 接入 OpenAI，让真人老师配 AI 生成课程摘要和练习；企业还能用 Oracle 云的已有承诺额度直接付费，连 Codex 也算进去。意思很明确：不必让用户每天想着打开 OpenAI，只要把 OpenAI 塞进培训、教育和云采购里。

第四条，安全线也在收紧。Google 起诉一个叫 Outsider Enterprise 的中国诈骗团伙，称对方用 AI 两周发出二百五十万条短信，诈骗数十万受害者；另一边，DeepMind 拿出一千万美元做 multi-agent 安全研究基金。一个是现实世界的 AI 诈骗，一个是提前补多 agent 的安全窟窿。AI agent 从玩具变成流程之后，安全就不再是加分项，而是入场券。

论文这边，第一篇 Arbor 问了个关键问题：agent 不是不会做研究，而是怎么把经验留住。它用一棵 Hypothesis Tree，把假设、实验、证据和结论都挂起来，让协调者跨轮吸收教训，再派执行者去隔离环境里验证。结果六个真实研究任务全拿最佳，平均相对增益是 Codex 和 Claude Code 的二点五倍以上。换句话说，长程 agent 的记忆，不能只塞在上下文里，得变成外置的长期资产。

第二篇 RACES 更像是在给 RL 训练搭乐高。可验证环境很珍贵，因为它能自动判对错，但人工搭环境太慢。RACES 把五十个基础环境按输入输出类型递归组合，效果约等于三百个独立环境，在六个没见过的 benchmark 上让十四 B 模型平均涨三分左右。点数不是最惊人的，真正重要的是方向：当模型和算法越来越像，谁能更快合成环境，谁就更有扩展空间。

所以今天的关键词不是模型更大，而是边界更清楚。Fable 5 被拉闸，agent 烧账单，诈骗团伙批量发短信，论文开始研究长期记忆和环境复用。AI 正从聊天窗口走进真实世界，而真实世界会问它三个问题：谁授权，谁买单，谁负责。我是柿子树下的猫wanjeans，我们明天见。
