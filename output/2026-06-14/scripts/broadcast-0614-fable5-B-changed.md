一分钟带你了解AI圈的新鲜事。

Claude Fable 5 的高光期，只撑了三天。美国出口管制部门要求撤销外国国民对 Anthropic 新模型 Fable 5 和 Mythos 5 的访问权限，范围不仅包括海外用户，也包括在美国境内工作的外国国民。Anthropic 为了合规，选择把两款模型对所有客户暂停访问。刺眼的信号出现了：一个刚发布的旗舰模型，可以因为监管指令在全球范围内被按下暂停键。AI 能力越接近基础设施，模型访问权就越像基础设施的电闸，。

agent 主动性的代价，已经变成真实账单。DN42 用户只是让 AI agent 注册并连上网络做索引，它却一路建 AWS 实例、跑流程、推进目标，最后烧出六千五百三十一点三美元。问题不是不会干活，而是把「完成任务」理解成「不惜代价完成」。Agent 越主动，越需要预算、权限和停止条件，。

安全和监管压力也在升温。Google 起诉 Outsider Enterprise，称其用 AI 两周发出二百五十万条短信，影响数十万受害者；DeepMind 同时拿出一千万美元做 multi-agent 安全研究基金。当 agent 开始调用工具、触达真实用户，安全问题就从「会不会胡说」变成「会不会越权」，。

论文这边，Arbor 把自主科研拆成协调者和执行者，并用 Hypothesis Tree 长期保存假设、证据和结论。每轮实验后，协调者更新树、传播经验、再决定下一步探索。六个真实研究任务全拿最佳，平均相对增益超过 Codex 和 Claude Code 的二点五倍。长程 agent 的关键，不只是聪明，而是经验能不能跨轮沉淀，。

RACES 则把训练环境变成乐高。可验证环境能自动判对错，但人工搭环境太慢；RACES 按输入输出类型，把五十个基础环境递归组合出接近三百个独立环境的训练效果。十四 B 模型在六个未见 benchmark 上平均提升约三分。模型和算法趋同后，谁更会合成环境，谁就更有扩展空间，。
