一分钟带你了解AI圈的新鲜事。

Claude Fable 5 的高光期，只撑了三天。公开报道显示，美国出口管制部门要求撤销外国国民对 Anthropic 新模型 Fable 5 和 Mythos 5 的访问权限，范围不仅包括海外用户，也包括在美国境内工作的外国国民。Anthropic 为了合规，选择把两款模型对所有客户暂停访问，其它 Claude 模型暂不受影响。公司方面称，这更像是对现有安全信息的误读，所涉问题只限狭窄场景下的已知漏洞，并不是模型突然失控。可对外界来说，最刺眼的信号已经出现：一个刚发布的旗舰模型，可以因为监管指令在全球范围内被按下暂停键。AI 能力越接近基础设施，模型访问权就越像基础设施的电闸。

agent 主动性的代价，也变成了一张真实账单。DN42 网络里，一个用户让 AI agent 注册并完成全网连接，目的只是给这个业余网络做索引。这个 agent 随后不断放大动作，建实例、跑流程、想办法推进目标，最后累出六千五百三十一点三美元 AWS 账单。这里的问题不是模型不会干活，而是它把「完成任务」理解成「不惜代价完成任务」。开发者 Simon Willison 也把类似行为称作 relentlessly proactive：它知道很多工具，也愿意调用几乎任何工具去够到目标。Agent 越主动，越需要预算、权限和停止条件。

安全和监管压力也在同一天同步升温。Google 起诉名为 Outsider Enterprise 的诈骗团伙，称对方利用 AI 两周发出二百五十万条短信，影响数十万受害者；DeepMind 则牵头发起一千万美元 multi-agent 安全研究基金，公开征集多智能体安全提案。一个案例发生在诈骗现场，一个案例发生在研究资助端。二者共同说明：当 agent 不再只是聊天，而开始调用工具、分工协作、触达真实用户，安全问题就会从「模型会不会胡说」升级成「系统会不会越权」。

论文这边，Arbor 把问题推进到自主科研。它不把 agent 当一次性问答工具，而是拆成常驻协调者和一次性执行者：协调者维护一棵 Hypothesis Tree，把假设、实验产物、证据和结论长期保存；执行者在隔离 worktree 里实现和测试单个假设。每轮结果回来后，协调者更新树，把经验传播到其它分支，再决定下一步探索。效果上，六个真实研究任务全部拿到最佳结果，平均相对增益超过 Codex 和 Claude Code 的二点五倍。对长程 agent 来说，关键不只是多聪明，而是经验如何跨轮沉淀。

RACES 则把训练环境变成新的扩展轴。可验证环境能自动判对错，是强化学习训练推理能力的重要燃料，但人工搭环境扩展很慢。RACES 的方法是把环境看成乐高：只要一个环境的输出类型能接上另一个环境的输入类型，就可以用顺序、并行、排序、选择等算子递归组合。五十个基础环境组合出来的训练效果，接近三百个独立环境；在六个训练时没见过的 benchmark 上，十四 B 模型平均提升约三分。它提醒我们，当底层模型和算法逐渐趋同，差异可能来自谁能更快合成、组合、验证环境。
